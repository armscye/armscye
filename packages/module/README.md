# @armscye/module

> A collection of shared standard TypeScript definitions.

## Installation

Using npm:

```sh
npm install --save-dev @armscye/module
```

or using yarn:

```sh
yarn add @armscye/module --dev
```

## Interfaces

### Module

```ts
export interface Module {
  /**
   * Register configurations.
   */
  register(): any;
}
```
